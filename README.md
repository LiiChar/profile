# Personal Portfolio

My personal portfolio and resume site. Built entirely with modern React ecosystem.

## Features
- Next.js 14 App Router + Server Components
- 3D animations (Framer Motion + Spline)
- Dark/Light theme
- 100/100 Lighthouse (Performance, Accessibility, SEO)

## Tech Stack
- Next.js 14 + TypeScript
- Tailwind CSS
- Framer Motion
- Vercel
